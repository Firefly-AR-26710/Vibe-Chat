# 标签分类 (Triage Labels)

各项技能均采用五种标准的分流角色（Triage roles）来进行沟通。本文件将这些核心角色映射到本项目 Issue Tracker 中实际使用的标签字符串。

| Agent 技能中的标签名称 | 本项目中实际使用的标签 | 含义说明 |
| -------------------------- | -------------------- | ---------------------------------------- |
| `needs-triage`             | `needs-triage`       | 待评估：需要项目维护者进行评估 |
| `needs-info`               | `needs-info`         | 待补充信息：正在等待问题报告者提供更多信息 |
| `ready-for-agent`          | `ready-for-agent`    | 需求明确：已经详细说明，准备好由 AI Agent 在后台自动处理 |
| `ready-for-human`          | `ready-for-human`    | 需要人工：需要人类开发者介入实现 |
| `wontfix`                  | `wontfix`            | 不予处理：拒绝修复或采取行动 |

当某项技能提到某个角色时（例如“应用 AFK-ready (可由AI后台处理) 标签”），请使用本表中对应的实际标签字符串。
