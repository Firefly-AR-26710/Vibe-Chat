# 领域词汇表 (Context & Glossary)

## 核心业务概念

- **情绪标签 (Emotion Label)**: LLM 根据用户输入文本解析出的离散分类状态（如：焦虑、悲伤、激动、平静等）。
- **情绪强度 (Intensity Score)**: 与情绪标签关联的数值（如 1-10），用于衡量该情绪的强烈程度。
- **匹配队列 (Match Queue)**: Redis 中按“情绪标签”和“强度区间”划分的等待池，用户提交状态后会进入对应的队列寻找伴侣。
- **匿名会话 (Anonymous Room)**: 匹配成功后创建的 WebSocket 房间。参与者身份通过系统生成的临时标识隔离。
- **AI 伴侣 (AI Companion)**: 在规定时间内（匹配超时）未能在队列中找到真实匹配用户时，由 LLM 扮演的兜底聊天对象，提供与用户当前情绪相呼应的陪伴。
- **情绪分析工作流 (Emotion Analysis Graph)**: 基于 LangGraph 构建的后台决策流程，负责统筹文本分析、路由判断和 AI 伴侣的会话生成。
