## Agent skills (AI 技能配置)

### 问题追踪 (Issue tracker)
本项目的所有需求和任务均在 GitHub Issues 中进行管理。详见 `docs/agents/issue-tracker.md`。

### 标签分类 (Triage labels)
本项目使用默认的核心状态标签（如 needs-triage, ready-for-agent 等）。详见 `docs/agents/triage-labels.md`。

### 领域文档 (Domain docs)
本项目采用单上下文（Single-context）结构，业务术语和架构决策统一存放在根目录的 `CONTEXT.md` 和 `docs/adr/` 中。详见 `docs/agents/domain.md`。
